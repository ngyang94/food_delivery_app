# Food delivery app

a demo food delivery app build using expo with hardcoded data, built by refering to video from youtuber called "JavaScript Mastery"

## Setup

1. download repo
2. open terminal and navigate to the directory and type "npm install"
3. type in "npx expo start" to start the project
4. open expo go app and scan the QR show on the screen to run the app

## Usage

- login user, with hardcoded email="ng@email.com" and password="ng123"
- register new user
- search for food by keyword or category
- add, edit and remove food order to cart
- place food order
- edit profile
